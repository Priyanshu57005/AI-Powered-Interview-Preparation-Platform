# AI Interview Preparation Platform

Built a full-stack web application using React, Node.js, Express, and MongoDB.
Integrated Gemini AI to generate role-specific interview questions and feedback.
Implemented JWT authentication, protected routes, PDF report generation, and resume uploads.
Added interview analytics dashboard and automated report sharing.

## Tech Stack

Frontend
- React
- Vite
- React Router
- Axios

Backend
- Node.js
- Express.js
- MongoDB
- JWT
- bcrypt

AI
- Google Gemini API
